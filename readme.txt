=== Auto Twitter Hashtags ===

Plugin Name: Auto Twitter Hashtags

Author: Kayleigh Thorpe

Version: 2.0

Author URI: http://albinofruit.com

== Description ==

Auto Twitter Hashtags is a WordPress plugin that reads the content of your posts and pages for hash-tagged words and links them to the relevant search entry in Twitter.



== Installation ==

1. Upload The plugin's zip file to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
